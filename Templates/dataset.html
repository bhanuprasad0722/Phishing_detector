About the Dataset

Our dataset includes a wide variety of URLs categorized as phishing or legitimate, extracted from reliable sources. It contains numerous features like SSL information, domain age, and URL patterns crucial for machine learning models.

Key Features:

  • Over 10,000 labeled records
  • Detailed URL-based features
  • SSL Certificate Analysis
  • Domain Registration Length
  • Third-party Website Check
  • Presence of '@', '-', and multiple subdomains
